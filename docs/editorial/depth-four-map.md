# Depth-Four Map

This is a planning skeleton for eventual depth-four coverage. It does not add nodes, edges, public prose or sources.

## Scope And Purpose

Use this document to plan coherent coverage across the atlas before implementing public nodes. The map should help editors compare route shape, candidate strength, duplicate risks, source needs and shared-node opportunities.

## Definition Of Coverage Depth

Coverage depth is the shortest approved route from Bitcoin to a canonical node. It is an editorial planning measure, not a public claim about where a node truly belongs.

A shared node may appear at several Path depths depending on the route travelled. Coverage depth should never be stored or presented as the node's only valid position.

## Proposed Node Budget

Initial planning range:

- approximately 120-180 canonical nodes across the eventual depth-four atlas
- subject to human pruning
- not a target that justifies filler

## Approximate Node Budget

Use the proposed node budget as a pressure check, not as a quota. Stop routes naturally when further subdivision would create weak bubbles, duplicated concepts or unsupported prose.
